# New File Templates v1.0.0

**Release Date:** 2026-01-18

Create new notes with dynamic, template-driven file names using date formats, counters, and custom template syntax.

---

## Features

### Template Management
- Create, edit, and manage multiple title templates
- Drag-and-drop reordering of templates
- Searchable template selection with fuzzy search

### Multiple Access Points
- Command palette: "Create New Templated File"
- Per-template commands for quick access
- Folder context menu integration
- Ribbon icon for one-click access
- Full mobile support (iOS and Android)

### Dynamic Variables
Built-in variables for title patterns:
- `{{date}}`, `{{time}}`, `{{datetime}}` - Current date/time with customizable formats
- `{{timestamp}}` - Unix timestamp
- `{{year}}`, `{{month}}`, `{{day}}` - Individual date components
- `{{counter}}` - Auto-incrementing number based on existing files
- Custom formats supported: `{{variable:FORMAT}}` using moment.js syntax

### Interactive Prompts
- `{% Prompt Name %}` - Request user input when creating files
- `{%? Optional Prompt ?%}` - Optional input fields
- Multiple value types: text, numeric, date, time, datetime
- Built-in format presets and custom moment.js formats
- Inline syntax configuration: `{% Name:type:format %}`

### Syntax Highlighting & Autocomplete
- Visual highlighting for template syntax in editor and settings
- Smart autocomplete for variables, prompts, types, and formats
- Moment.js token suggestions with live preview examples
- Context-aware token ordering

### File Template Integration
- Apply content templates to new files
- Variable substitution in content: `{{title}}`, `{{date}}`, `{{time}}`
- Prompt syntax works in file content
- Templater plugin integration with auto-detection

### Additional Features
- Automatic filename sanitization (handles special characters)
- Configurable default folders per template
- "Current folder" targeting option
- Conflict handling for duplicate filenames
- "Enter File Prompts" command for processing prompts in existing files

## Installation

1. Open Obsidian Settings
2. Navigate to Community Plugins
3. Search for "New File Templates"
4. Click Install, then Enable
